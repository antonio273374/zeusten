## O que é o Mente Aberta?
 o mente aberta é um site, que visar a apridizagem de alunos de ensino medio, em fisica e matematica. Junto com professor zeuinstein. A partir deste site, é possível:
 - Personalizar o método de ensino
 - Auxílio em exercícios
 - Desevnolvimento de raciocínio lógico

## Linguagens de Programação
- JAVASCRIPT
- HTML
- CSS

## Público alvo
- Estudantes que apresentam difculdade nas matéiras de Matemática e física
- Estudantes que estão se preparando para concursos

## Pontos Positivos

- Aprendizado no próprio Ritmo: O estudante pode acessar ao site a qualquer momento e não depender de horários programados que pode ocasionar a perda de agum assunto
- Respostas rápidas e detalhadas: As respostas chegam ao usuário em tempo real e de maneira didática, visando uma explicação de fácil compreensão
